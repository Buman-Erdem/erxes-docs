# Header Description
Expandable description, shows image and description when expanded but just title when it's not.

## **Example**

```tsx
<>
	<HeaderDescription icon="https://erxes.io/static/images/logo/logo_dark.svg" title="Title" description="Description" />
</>

```

## **API**

```tsx
import HeaderDescription from "erxes-ui/lib/components/HeaderDescription";
```
* required prop

| NAME | TYPE | DEFAULT | DESCRIPTION |
| ---| --- | ---| --- |
| icon*	 | string |  | Shows image before title |
| title* | string |  | Define title of description |
| description*	 | string| | Define description |
