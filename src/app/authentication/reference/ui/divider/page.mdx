# Text Divider
Add text divider with text and date to divide contents.

## **Example**
Hover over the time to see date.

```tsx
<TextDivider text="Text divider" date="Sun Nov 19 2023 18:32:07 GMT+0800 (Ulaanbaatar Standard Time)" />
```

## **API**

```tsx
import TextDivider from "erxes-ui/lib/components/TextDivider";
```
* required prop

| NAME | TYPE | DEFAULT | DESCRIPTION |
| ---| --- | ---| --- |
| text*	 | string |  | Shows text upper side of text divider |
| date*	 | Date |  | Shows date |
