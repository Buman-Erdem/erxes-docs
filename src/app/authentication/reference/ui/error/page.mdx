# Error message
Provide error message to indicate an error that occurred.

## **Example**

```tsx
<ErrorMsg>This is error</ErrorMsg>
```

## **API**

```tsx
import ErrorMsg from "erxes-ui/lib/components/ErrorMsg";
```
* required prop

| NAME | TYPE | DEFAULT | DESCRIPTION |
| ---| --- | ---| --- |
| children*	 | string |  | Declare error message |
