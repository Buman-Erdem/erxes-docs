# Chip
## **Examples**
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/edd05264-cb2e-4a49-7f8e-419bca5eb200/public)
```tsx
<Chip>chip</Chip> 
```

## **Capitalized**
Make the first letter of chip text capital by adding `capitalize`
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/e0b7e578-a3c8-4d52-0a8a-460b686bc600/public)
```tsx
<Chip capitalize >chip</Chip> 
```

## **Front Content**
Add an extra content in front of the chip by `frontContent` prop.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/f7e7eb1b-9beb-4490-e18f-b9771a7d5400/public)
```tsx
<Chip frontContent={<Icon icon='envelope-alt' />} >chip</Chip> 
```

## **API**
```tsx
import Chip from "erxes-ui/lib/components/Chip";
```
* required prop

| NAME | TYPE | DEFAULT | DESCRIPTION |
| ---| --- | ---| --- |
| capitalize | boolean | false | Make the first letter of chip text capital |
| onClick* | function |  | Define click handler for cancel icon |
| children*	 | React.ReactNode	| | Shows chip text |
| frontContent | string	 |  | Add an extra content in front of the chip |
